# Tree Map

## Installation

```bash
npm install @dpc-sdp/yourvic-treemap --save
```

## Import

```javascript
import { TreeMap } from '@dpc-sdp/yourvic-treemap'
```

## Usage

```html
<tree-map
  :data="data" :valueAttr="value" :labelAttr="name"
/>
```
