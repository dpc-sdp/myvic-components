# Line Chart

## Installation

```bash
npm install @dpc-sdp/yourvic-line-chart --save
```

## Import

```javascript
import { LineChart } from '@dpc-sdp/yourvic-line-chart'
```

## Usage

```html
<line-chart
  :data="data"
/>
```

## Props

<Props of={ LineChart } />

### Data Format
The data object that should contain the following keys:
- labels: an array of values for one of the axes. In the case of a horizontal line chart this would be the Y-axis
- datasets: an array of objects (minimum 1 required - maximum of 4) that contain the following keys:
  - label (String): a label for this dataset
  - data (Array): the values for this dataset. The length of this should match the length of the labels key
